Fitt's Law- Pointing Tasks with Eye-Gaze

Idea Summary

ISO 9241 is a standard Pointing task which is used as the standard metric to evaluate pointing devices. It combines speed and accuracy in a single measure.We developed a pointing task which follows the ISO standard based on Fitt's Law and provides throughput(index of performance) of user as result.We mimicked the ISO 9241 Pointing task which can be used with Eye Gaze which can be installed as a Windows 10 App and a WebApp.

Description

We analyse the cursor movement trajectories using the metrics defined by MacKenzie [2001] with regards to the task axis (the line between starting source and intended target). These metrics look at the variability of movements as well as the number of events relating to cursor movement along the task axis towards the intended target.

We designed the application in a way, that allows native install in Windows 10 UWP app. No separate gaze library is required. The application outputs a log file on completion of task which can be used to assess the through of user. Also a Web App version is available which also helps visualise the pointer movement metrics.

Gaze controlled applications for users with Severe Speech and Motor Impairment(SSMI)

We digitise and develop intelligent applications for people with Severe Speech and Motor Impairment(SSMI) with Eye-Gaze as input modality.

Idea Summary

Due to spasticity in children with Cerebral Palsy, they could not use any assistive technologies like track ball, push button switch and so on. Therefore, the child's gaze is the one which he/she can voluntarily control. Thus, use of eye gaze tracking to improve literacy of differently abled children is of great interest.

Description

We investigate design constraints for developing graphical user interfaces in limited mobility environments, particularly to design and develop gaze-controlled interfaces for users with SSMI. These people are classified as Level 5 on Gross Motor Function Classification System(GMFCS), i.e. they cannot move without wheelchair. It is very challenging to investigate their response to visual stimuli, examine visual search pattern and area of interest on screen with eye gaze as input.We designed and developed Augmentative and Alternative(AAC) interfaces which uses eye gaze as input modality. Some examples of interfaces which you might find below are: Virtual Keyboard,Stored-Message Interface (Assistive Communication Board), Chat Application, Webcam based eye gaze control.
